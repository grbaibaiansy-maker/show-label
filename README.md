# 字符网格标注与对照

定义行列网格，通过拖动网格或图像完成对齐标定；用户逐字选择目标字符（标为 False），提交后与预先给定的真实 False 位置同屏对照（红框为用户标注，绿框为真实标签）。

## 使用方式

1. 用浏览器打开 `index.html`，或本地起静态服务后访问（如 `python3 -m http.server 8080`）。

2. **加载字符图像**：点击「加载字符图像」上传一张按行列排布的字符图。

3. **定义网格**：在「行数」「列数」中填写网格的行列数（如 4 行 5 列），系统按等分得到每个格子的长宽。

4. **对齐标定**：若网格与图像上的字符未对齐，可切换模式后拖动以对齐：
   - **拖动网格**：点击「拖动网格」后，在图像上按住鼠标拖动，网格整体移动，与图像内容对齐。
   - **拖动图像**：点击「拖动图像」后，在图像上按住鼠标拖动，图像整体移动，与网格对齐。
   - **重置对齐**：点击「重置对齐」将网格偏移和图像偏移恢复为 0。
   - 对齐完成后切回「选择字符」进行标注。

5. **逐字选择**：在「选择字符」模式下，点击图像上的格子可选中/取消（红框 + 标签 "False"）。右侧列表显示已选字符的 (行, 列) 索引。

6. **填写真实 False 位置**：在「真实 False 位置」中按行输入真实为 False 的字符位置，每行一个，格式为 `行,列`（如 `0,1`）。示例：
   ```
   0,1
   1,2
   2,3
   ```

7. **提交**：点击「提交」后，在对应格子绘制**绿色矩形框**，与用户的**红色标注**同屏对照。

8. **清空选择**：点击「清空选择」可清除所有用户标注并隐藏绿框。

## 图例

- **红框**：用户标注（标签统一为 False）
- **绿框**：真实 False 字符的行列位置（提交后显示）
